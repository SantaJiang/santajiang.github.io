将Hexo迁移至MacOS

2021-08-02 博客搭建 Views

当时用的Windows搭建的这个博客，现在要迁移到我的MacBook Pro上来，在查了很多资料后动手实践，发现还是比较方便的，现写下流程

Node.js / Git

首先要安装Node.js与Git的环境，这里省略就不多说了

复制文件

将原Windows电脑上的博客复制如下的文件，并暂存起来：

安装Hexo

打开终端，当前路径均可，默认只会装在固定的位置
执行sudo npm install -g hexo-cli

初始化博客

在你想要放置博客的位置，然后操作：

打开终端
执行hexo init blog

博客迁移

将之前暂存的文件，全部替换到刚才创建的blog文件夹中，然后操作：

打开终端
执行npm install
执行hexo g
执行hexo d
查看效果

至此博客就成功迁移到MacOS中了，后续的操作与Windows下无差别
