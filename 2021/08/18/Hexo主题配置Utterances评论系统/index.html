Hexo主题配置Utterances评论系统

Posted on 2021-08-18 Edited on 2021-08-24 In 搭建博客 Views:

在网上查找这部分教程的时候，很多教程已经过时了，因为最新版的Next默认已经不内置gitment评论系统了，我在多次配置后选用了目前Next 8+版本内置的Utterances，效果不错，配置起来相较于其他的评论系统简单很多，现写下配置流程

创建Github仓库

需要在自己的 Github 账号下创建一个仓库来存放评论，创建的仓库名最好为utterances，方便后续自己识别，注意仓库为public

安装Utterances

安装地址：https://github.com/apps/utterances

访问安装地址，进入安装页面，单击 Install 按钮

接下来是一个权限的选择，可选可访问全部仓库，或者指定刚才创建的仓库
